PROVEE

Description

One of the goals of PROVEE (Progressive Exploration of Embedding Spaces) is to incrementally visualize embeddings. Using the incremental approach in visualization, it is easier to hold large embedding files into memory and the user will be able to extract useful information already during the process of projection before it is finished. Project

One of the core features is that custom visualiser, specific for the use-case embedding can be implemented in the tool. One of these example embeddings is shown below. this image shows how a successful BERT (Bidirectional Encoder Representations from Transformers) model learns during training on a language task. Each dot represents data from one of the model’s layers at different training steps, with colors indicating the layer number. Early in training, some upper layers begin to shift and form tighter clusters, which suggests they are learning useful features. Over time, the points continue to move and change, showing that the model is actively learning and refining its internal representations Furthermore, the details of this project are thoroughly described in the paper listed below.
